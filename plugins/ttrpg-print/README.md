# ttrpg-print

Production print-ready : design typographique, prompts Midjourney pour illustrations JDR, audit qualité de prompts, cartes de tarot.

## Skills

| Skill | Rôle |
|---|---|
| `ttrpg-print-design` | Layout, typographie, specs print pour DriveThruRPG |
| `midjourney-prompt-generator` | Génération de prompts Midjourney pour illustrations JDR |
| `midjourney-prompt-auditor` | Audit qualité de prompts avant dépense de crédits (7 vérifications structurelles) |
| `tarot-card-portrait` | Prompts pour portraits de personnages style carte de tarot |

## Installation

```
/plugin install ttrpg-print@ludomancien-skills
```

## Workflow type

1. `midjourney-prompt-generator` produit un prompt pour ton illustration
2. `midjourney-prompt-auditor` audite le prompt avant que tu dépenses des crédits
3. Une fois les visuels en main, `ttrpg-print-design` t'aide à les intégrer dans la mise en page
