# 5e 2014 → 5e 2024 Deltas

Complete mechanical delta between D&D 5e 2014 (PHB / DMG / MM
2014–2023 with errata) and D&D 5e 2024 (PHB / DMG / MM 2024).

Most 5e 2014 content survives unchanged. This document focuses on
what changed.

---

## Format-Level Changes

### Stat Block Format

| 2014 | 2024 |
|---|---|
| `bonus action` (lowercase) | `Bonus Action` (capitalized) |
| Generic Multiattack line | Restructured action block with `Strength` / `Dexterity` saves explicit |
| "Languages —" line | "Languages" line, sometimes adds telepathy specifics |
| Implicit `proficiency bonus` field | Explicit "Proficiency Bonus +X" field |
| Free-form challenge rating line | Standardized: `CR X (XP / PB +X)` |
| `Habitat` not listed | `Habitat: [biome]` field |
| `Treasure` not listed | `Treasure: [tier]` field |

### Vocabulary

| 2014 | 2024 |
|---|---|
| Race | Species |
| Racial trait | Species trait |
| Subrace | Lineage / variant species |
| Lawful Good / Chaotic Evil (mandatory) | Alignment optional; "Typically [alignment]" guidance |
| Spellcaster Level | Spell Slot Level (where relevant) |
| Charisma (Persuasion) check | Persuasion check (Charisma) |
| Touch range | Touch (Range 5 ft) |
| HD = Hit Die | HD = Hit Die (unchanged), HP roll method unchanged |

## Core Rules Changes

### Initiative

- **2014:** Roll Dexterity + initiative bonus. Ties broken by Dex
  score, then DM call.
- **2024:** Same baseline, but tie-breaking is explicit (Dex
  score, then DM call). Some classes / features grant fixed
  initiative.

### Weapon Mastery

- **2014:** No weapon mastery system. Weapons distinguished by
  damage type and properties (Versatile, Finesse, Reach, etc.).
- **2024:** Weapon Mastery system added. Each weapon has a mastery
  trait (Cleave, Graze, Nick, Push, Sap, Slow, Topple, Vex). Class
  features grant access to weapon masteries (Fighter gets several
  at level 1, Rogue starts with one, etc.).
- **Conversion impact:** Combat encounters need re-validation. A
  fight balanced for 5e 2014 Fighters can become significantly
  easier when those Fighters access Push (5 ft push every hit) or
  Sap (disadvantage on next attack).

### Hide Action

- **2014:** Stealth is contextual. Players make Dex (Stealth)
  checks vs passive Perception or active rolls.
- **2024:** Explicit Hide action with DC 15 default (or higher if
  the GM rules so). Hide creates the *Invisible* condition until
  noise / attack / revealing.
- **Conversion impact:** Stealth-based encounters need DC explicit.
  Default to DC 15 unless source specifies otherwise.

### Search Action

- **2014:** Implied; the GM decides whether searching takes time
  or a check.
- **2024:** Explicit Search action with Wisdom (Perception) check.
  Some abilities specifically use Search.
- **Conversion impact:** Distinguish between "passive Perception
  to notice X" and "active Search action for DC X".

### Unarmed Strike

- **2014:** Simple weapon attack: melee, 1 + Str damage,
  bludgeoning.
- **2024:** Choose Damage (1 + Str modifier bludgeoning), Grapple,
  or Shove on each unarmed strike. Distinct options per use.
- **Conversion impact:** Grappler / brawler stat blocks should
  reflect 2024 options.

### Exhaustion

- **2014:** 6 exhaustion levels with cumulative effects (1: dis.
  on ability checks; 2: half speed; 3: dis. on attacks/saves; 4:
  HP max halved; 5: speed 0; 6: death).
- **2024:** Simplified — exhaustion penalty per level: -2 to d20
  rolls and -5 ft speed per level, accumulating. At level 6,
  character dies.
- **Conversion impact:** Stat blocks that grant exhaustion need
  re-validation (a 2014 "1 level of exhaustion" effect is
  arithmetically different from 2024).

### Inspiration

- **2014:** Inspiration token granted by DM; expend for advantage
  on one roll.
- **2024:** Heroic Inspiration (renamed). Some species, classes,
  and feats grant inspiration mechanically. Mechanically similar
  but distribution is more systematic.

## Class Changes

### Cleric

- **Channel Divinity:** 1/short rest at level 2 (2014) → 3/short
  rest at level 17 (2024 has more uses earlier; Channel Divinity
  options expanded).
- **Divine Order:** New level-1 feature; choose Protector or
  Thaumaturge sub-path.
- **Domain spells:** Some Domain lists updated (e.g., Life Domain
  changed slot allocations).
- **Conversion impact:** Cleric NPCs / monsters with Cleric class
  levels need refresh.

### Fighter

- **Weapon Mastery:** Fighter gets 3 weapon masteries at level 1
  (up to 5 by level 9). Major buff.
- **Action Surge:** Same uses but timing clarified.
- **Subclasses:** Champion, Battle Master, Eldritch Knight,
  Psi Warrior — most updated; Battle Master's maneuver pool
  refreshed.
- **Conversion impact:** Fighter NPCs benefit from mastery
  trait additions.

### Wizard

- **Subclasses:** Renamed in some cases (e.g., School of
  Necromancy adjusted). Subclass features at different levels.
- **Spell choices:** Wizards prepare spells more flexibly in 2024.
- **Conversion impact:** Spell-list-NPCs unchanged unless they
  invoke specific subclass features.

### Other classes

- **Sorcerer:** Sorcery Points + Metamagic structure refined;
  Innate Sorcery Bonus Action grant.
- **Barbarian:** Rage mechanics refined; Reckless Attack
  unchanged.
- **Bard:** Bardic Inspiration timing clarified; Magical Secrets
  shifted.
- **Druid:** Wild Shape stat blocks changed (mechanical-only
  changes; flavor preserved).
- **Monk:** Discipline Points instead of Ki Points (rename).
- **Paladin:** Lay on Hands still works; Smite spells reworked
  (Banishing Smite, Wrathful Smite, etc. — some rebalanced).
- **Ranger:** Hunter's Mark always-prepared and a free casting
  per long rest in some builds.
- **Rogue:** Cunning Strike option (use Sneak Attack damage to
  impose conditions).
- **Warlock:** Pact Magic + Mystic Arcanum + Eldritch Invocations
  refined; some Invocations renamed.

## Spell Changes

### Renamed or Reworked

| 2014 | 2024 |
|---|---|
| Spiritual Weapon | Same name, same mechanic; minor rebalance |
| Hunter's Mark | Always prepared for Rangers; free once per long rest |
| Hex | Restructured; free once per long rest for Warlocks |
| Conjure Animals / Beasts / Elementals | Replaced by `Summon Beast` / `Summon Elemental` pattern (one statblock scales) |
| Power Word Kill | HP cap raised to 100 (same trigger) |
| Banishing Smite | Restructured |
| True Polymorph | Same |
| Fireball | Same (8d6 baseline) |

### Dropped or Deprecated

Some 2014 spells dropped or deprecated in 2024 (verify with PHB
2024 spell list). Common removals: redundant spells, spells
mechanically equivalent to others.

### Conversion Protocol for Spells

1. If the spell exists in 2024 with same name: use 2024 version.
2. If renamed: cite 2024 name; mention old name in conversion
   notes for legacy readability.
3. If dropped: find closest 2024 equivalent; if none, custom-
   build via `spell-creator`.

## Monster Changes

### General Changes

- **Stat block layout:** Updated to 2024 format (see Format-Level
  Changes above).
- **CR:** Most monsters at same CR. Some rebalanced (e.g.,
  certain dragons, certain undead). Verify per-monster.
- **Habitat field:** All 2024 monsters list habitat (Arctic,
  Coastal, Desert, Forest, Grassland, Mountain, Swamp,
  Underdark, Underwater, Urban, Planar).
- **Treasure field:** All 2024 monsters list treasure tier
  (None, Individual, Hoard).

### Specific Monster Adjustments

| Monster | 2014 → 2024 Change |
|---|---|
| Goblin | Stat block tightened; Nimble Escape stays |
| Bugbear | Action economy reformatted; surprise attack mechanic clarified |
| Hobgoblin | Disciplined Strike upgraded |
| Orc | Aggressive trait refined |
| Dragons (Adult/Ancient) | Lair actions / regional effects revised; legendary actions clarified |
| Lich | Spellcasting reflected as innate (with rest recovery); CR slightly adjusted |
| Beholder | Eye ray mechanics refined; central eye antimagic cone clarified |
| Mind Flayer | Mind Blast wording updated |

(This is illustrative. Verify the specific monster in MM 2024.)

### Conversion Protocol for Monsters

1. **Found in MM 2024:** use 2024 version directly. Cite page.
2. **Not in MM 2024 but in 5e 2014:** convert format
   (vocabulary, capitalization, layout); leave mechanics intact;
   add Habitat + Treasure fields.
3. **Homebrew:** convert format; verify CR math against 2024 CR
   tables (which are slightly recalibrated but mostly compatible);
   add Habitat + Treasure fields.

## Magic Item Changes

### General Changes

- Most items unchanged.
- **Format updates:** Items list category, rarity, attunement
  rules clearly.
- **Some rebalancing:** A few iconic items (e.g., Cape of the
  Mountebank, certain Belts of Giant Strength) rebalanced.

### Conversion Protocol for Magic Items

1. **In DMG 2024:** use 2024 version.
2. **Custom:** apply format polish; cite rarity / category /
   attunement clearly.

## Background Reframing

### 2014

Backgrounds grant skill proficiencies, tool proficiencies,
languages, and a feature (Background Feature). PCs choose
appearance / personality traits / ideal / bond / flaw.

### 2024

Backgrounds grant:
- Two skill proficiencies (same as 2014).
- One tool proficiency (or language replacement).
- An **ability score increase** (1 ability +2, or 1 ability +2 + 1
  +1, or other distributions — selectable from background list).
- An **Origin Feat** (chosen at character creation; usually a
  modest feat like Magic Initiate, Skilled, Tough, Lucky).
- A **language**.

**Conversion impact:**
- 2024 PCs are mechanically stronger at level 1 (effective +1
  ability + a feat). Encounter calibration is roughly +1 effective
  level.
- Background-as-NPC-trait may need re-statting.

## Condition Changes

Most conditions unchanged. Two changes worth noting:

### Exhaustion

See above. Simplified.

### Invisible

- **2014:** A creature you can't see has advantage on attacks vs
  you; you have disadvantage on attacks vs them.
- **2024:** Plus: an Invisible creature is now also concealed
  (Hidden, with the Hide action interaction). Less restrictive
  on what counts as Invisible.

## Conversion Workflow Summary

For a 5e 2014 → 5e 2024 conversion:

1. **Update vocabulary.** Replace "race" with "species",
   capitalize "Bonus Action", etc.
2. **Update stat block format.** Add Habitat + Treasure fields,
   restructure action blocks to 2024 layout.
3. **Verify each monster.** Use MM 2024 versions where available;
   leave homebrew alone unless balance differs.
4. **Verify each spell.** Use PHB 2024 versions; map renames; flag
   dropped spells.
5. **Adjust encounter calibration.** PCs are ~+1 effective level
   in 2024 due to background ability scores + Origin Feat. Bump
   encounter difficulty one tier in calibration if challenge is
   important.
6. **Update Hide / Search / unarmed strike mechanics.** These
   changed enough to warrant rule-call updates in encounter notes.
7. **Update class-NPC builds.** Cleric Channel Divinity, Fighter
   weapon mastery, Sorcerer sorcery points, etc.
8. **Document changes.** Conversion notes appendix lists every
   non-trivial change.
