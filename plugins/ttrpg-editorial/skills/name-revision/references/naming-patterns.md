# Naming Patterns Reference

Techniques for generating distinctive, non-generic names.

## Core Principle

The best fantasy names feel both unfamiliar and pronounceable. They should have roots (real or implied) rather than sounding like "fantasy word soup."

## Technique 1: Real-World Etymology Mining

Draw from underused historical and regional sources:

### Underused Language Pools
- **Breton/Welsh**: Maret, Gwenvred, Yannig, Konan, Herve, Nolwenn
- **Basque**: Iker, Eneko, Amaia, Irati, Koldo, Ainhoa
- **Georgian**: Tamar, Nino, Giorgi, Vakhtang, Levan, Ketevan
- **Finnish**: Väinö, Aino, Toivo, Ilmari, Lempi, Kauko
- **Hungarian**: Attila, Zsolt, Ildiko, Eszter, Bence, Reka
- **Slavic (non-Russian)**: Vlastimil, Miroslav, Jaromir, Bozena, Milena, Dagmar
- **Old Norse (actual)**: Sigurd, Gudrun, Hrolf, Thyra, Leif, Aud
- **Arabic/Persian**: Rostam, Bahram, Soraya, Parisa, Dariush, Mitra
- **Old English**: Wulfric, Aldhelm, Eadgyth, Godgifu, Osric, Aelfled

### Method
1. Find names from the target culture/era
2. Modify slightly if needed (change one vowel, drop a syllable)
3. Ensure it doesn't appear on banned list

## Technique 2: Consonant-Heavy Construction

LLMs default to soft, vowel-rich names. Counter with:

### Strong Consonant Clusters
- Initial: Kr-, Gv-, Br-, Str-, Vr-, Zd-, Sv-
- Medial: -rk-, -lg-, -nd-, -xt-, -ft-, -mb-
- Final: -rt, -sk, -nd, -rm, -lt, -rn

### Examples
- Krastov, Gverda, Brunt, Strale, Vrekka, Zdena
- Holgrim, Mundrik, Torfin, Ragneld, Skarven

## Technique 3: Unexpected Vowel Patterns

Avoid the -ia, -ea, -ae patterns. Use:

- Single short vowels: Maret, Toril, Belen, Kavan
- Unexpected combinations: Eudice, Aoibhe, Oihane
- Back vowels (u, o): Urda, Osma, Kolbrun, Gudny

## Technique 4: Truncation and Erosion

Take a longer name and wear it down (as happens naturally in language):

- Aleksander → Sander → Sand
- Margarethe → Grete → Gret
- Theodoric → Thierry → Terric
- Bartholomew → Bartol → Bart

## Technique 5: Occupational/Descriptive Origins

Names that originally described someone:

- Varga (Hungarian: shoemaker)
- Kovacs (Hungarian: smith)
- Tessier (French: weaver)
- Kallio (Finnish: rock)
- Brennan (Irish: sorrow/raven)

Convert these into first names or modify them.

## Technique 6: Place Name Adaptation

Real places make grounded fantasy names:

- Cantal (French region) → Cantar
- Ossau (Pyrenees valley) → Ossan
- Kvarken (Baltic strait) → Kvaren
- Morvan (French massif) → Morv

## Technique 7: Sound Symbolism Subversion

LLMs associate certain sounds with certain traits. Subvert:

| Trope | Subversion |
|-------|------------|
| Soft names = feminine | Gretta, Brunhild, Ragnvi |
| Hard names = masculine | Ilar, Senne, Emeric |
| Sibilants = mysterious | Bolg, Krat, Mundric |
| -ia/-ara = exotic | Hild, Tord, Vigga |

## Place Naming Techniques

### Compound from Function
Not "Shadowmere" but what the place actually does:

- Trade town → Markthof, Stapelburg, Wharfton
- Mining → Grubenstadt, Pitford, Schachtheim
- River crossing → Furtbach, Bridgeholm, Vadum

### Mundane + Specific
Real places have boring, specific names:

- Trois-Rivières, Bois-Guillaume, Pont-l'Évêque
- Steinbach, Altendorf, Neukirchen
- Whitby, Grimsby, Thirsk

### Corrupted Historical
Take a real place name etymology and corrupt it:

- "Ford on the Avon" → Avonford → Anfort → Anfor
- "Clearing of Wulf's people" → Wulfington → Wulling → Vullen

## Quick Generation Method

1. Pick a real-world name from an underused culture
2. Change ONE element (vowel, consonant, or syllable)
3. Verify it's not on the banned list
4. Verify it's pronounceable
5. Use it

### Example Workflow
Need: Female protagonist, Eastern European feel
1. Source: Hungarian "Ildikó"
2. Modify: "Ildek" (drop feminine ending)
3. Check: Not banned ✓
4. Pronounce: ILL-dek ✓
5. Final: Ildek
