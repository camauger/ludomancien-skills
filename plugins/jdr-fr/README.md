# jdr-fr

Contenu francophone pour le JDR : articles de mots rares (jeuxderole.org/mots-rares), conseils GM style fanzine, vocabulaire littéraire.

## Skills

| Skill | Rôle |
|---|---|
| `mots-rares-jdr` | Articles de mots rares français pour jeuxderole.org/mots-rares — étymologie, usage, exemples d'application en JDR |
| `fanzine-advice` | Articles de conseil GM style fanzine (1000-2000 mots) — ton personnel, voix d'auteur, pas de listes à puces génériques |

## Installation

```
/plugin install jdr-fr@ludomancien-skills
```

## Note

Contrairement à la plupart des autres plugins, le contenu produit ici est **toujours en français**, y compris les SKILL.md eux-mêmes — c'est la nature du plugin.
